Parent Authority

If you set a parent authority, users will be able to log in using that authority as well as via SSO.

For example, you could set up LDAP authentication and have that be the parent of this authority. That means that users will be able to log in via Mahara's inbuilt login forms using their LDAP details, as well as via SSO from an external authentication.

You do not have to set a parent authority. If you do not, users using this authority will only be able to access Mahara by SSO.
