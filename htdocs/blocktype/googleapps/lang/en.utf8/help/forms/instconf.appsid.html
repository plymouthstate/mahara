How to embed?

Google Docs

Google Apps for Educational domains: Sharing of individual files is identical as on a regular Google Apps account.

Your institution's administrator may have set the Google Apps so that only members of your institution can view your shared files.
In that case, a viewer will be presented with a Google Apps login screen and upon logging into Google Apps, the embedded files will become visible in the page.

Documents

  1. Open Google Docs document
  2. Click Share and then Publish to the Web (upper right corner)
    OR if you use the new interface:
    Click Collaborate and then Publish to the web. You do not have to set any sharing options via the Share button in the upper right-hand corner.
  3. Click Start publishing button
  4. Copy Document link or Embed code
  5. Paste it to the GoogleApps block

Spreadsheets

  1. Open Google Docs spreadsheet
  2. Click Share and then Publish as a web page (upper right corner)
    OR if you use the new interace:
    Click Collaborate and then Publish to the web. You do not have to set any sharing options via the Share button in the upper right-hand corner.
  3. Click Start publishing button
  4. Copy link from texarea
  5. Paste it to the GoogleApps block

Presentations

  1. Open Google Docs presentation
  2. Click Share and then Publish / embed (upper right corner)
  3. Click Publish document button
  4. Copy link or embed code from textarea
  5. Paste it to the GoogleApps block

Collections

  1. In Google Docs click My collections or Collections shared with me (list on left side)
  2. Click and select the Collection that you wish to embed
  3. Click Sharing - Settings link (on right side)
    OR if you use the new interface:
    Click on the arrow to the right of the collection name and then on Share and Share again.
  4. Change Permissions to Public and save changes
  5. Copy link from textbox
  6. Paste it to the GoogleApps block

Google Docs Viewer

This functionality works like the preview functionality in Gmail. You can embed the file, which is directly accessible over the Internet. You just have to paste the complete URL (including the filename) on which the file is accessible to the GoogleApps block.

If you use the new interface, you must publish the document first:
Click Collaborate and then Publish to the web. You do not have to set any sharing options via the Share button in the upper right-hand corner.

Google Docs Viewer currently supports embedding of the following file types: PDF, TIFF, PPT, DOC, DOCX.

Note: The files you've uploaded to Mahara are not directly accessible, so you can't embed those files.

Google Calendar

  1. Open Google Calendar
  2. Click on the little triangle next to the calendar that you wish to embed.
  3. Click Calendar settings.
  4. On the Calendar details tab scroll down to Embed This Calendar.
  5. Customize the calendar apperance and than copy the embed code from textarea.
  6. Paste it into the GoogleApps block.

Google Maps

  1. Open Google Maps
  2. Click My Maps (My Places in the new interface) link on the left upper side
  3. Click the name of the map that you wish to embed
  4. Click on Link (the link icon in the new interface) link on right upper side
  5. Customize the apperance of the map and copy embed code
  6. Paste it to the GoogleApps block
Note: Use steps 2 and 3 only if you wish to embed your Google Map. If you wish to embed any Google Map, then you can skip steps 2 and 3.

Google Books

  1. Open Google Books
  2. Find the book that you wish to embed.
  3. Click on the book title.
  4. Click on Link link on right upper side. The link icon only appears if you are allowed to embed this book.
  5. Copy the embed code of the book.
  6. Paste it to the GoogleApps block.
