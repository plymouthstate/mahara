learningstyles.swf

learningstyles.swf is highly customizable flash movie/component for displaying VAK (Visual, Auditory and Kinesthetic type) learning styles piechart.

The FlashVars accepts the folowing variables:

  • color1 which represents the visual style color used in piechart and must be in one of the folowing two formats: #RRGGBB or RRGGBB, where RR stands for red color component, GG stands for green color component and BB stands for blue color component
  • color2 which represents the auditory style color used in piechart and must be in one of the folowing two formats: #RRGGBB or RRGGBB, where RR stands for red color component, GG stands for green color component and BB stands for blue color component
  • color3 which represents the kinesthetic style color used in piechart and must be in one of the folowing two formats: #RRGGBB or RRGGBB, where RR stands for red color component, GG stands for green color component and BB stands for blue color component
  • values which represents comma separated list of all 3 learning styles values, computed from survey in which person answers to series of statements. Valid values must be between 0 and 360. Remember: The sum of all 3 values must be 360 (as 360° in full circle) - see the example below.
  • legend which represents comma separated list of 3 values (letters or numbers) which will represent the legend of the piechart - see example below.

Example 1:

<param name="flashVars" value="color1=#ff0000&color2=#008000&color3=#0000ff&values=100,120,140&legend=V,S,G" />

Example 2:

...with default graph color

<param name="flashVars" value="values=40,120,200&legend=X,Y,Z" />

Example 3:

...with default legend and color set without # sign

<param name="flashVars" value="color1=#ff8080&color2=#80ff80&color3=#8080ff&values=30,80,250" />

Example 4:

...one of the values equals 0, all three together equals 360.

<param name="flashVars" value="values=45,0,315" />
